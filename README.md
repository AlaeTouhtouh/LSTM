# LSTM

## Description
Ce projet prédit le prix de clôture de l'action Microsoft (MSFT) en utilisant un réseau de neurones Long Short-Term Memory (LSTM). Le modèle se base sur les données historiques pour prévoir les prix futurs.

## Fonctionnalités
- Prétraitement et normalisation des données
- Visualisation des prix d'ouverture, de clôture et du volume de transactions
- Modèle LSTM pour la prédiction des séries temporelles
- Entraînement et test avec visualisation des prédictions

## Prérequis
- Python 3.10 ou supérieur
- TensorFlow
- NumPy
- Pandas
- Matplotlib
- Seaborn
- scikit-learn
